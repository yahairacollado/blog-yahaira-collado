  • CLIvia generator

    CLIvia is a trivia game that presents the player with 10 randomly selected questions sourced from an API. After the user provides an answer, the program compares it with the correct answer obtained from the API and provides a response indicating whether the answer was correct or incorrect. In the case of an incorrect answer, the program reveals the correct answer to the user. If the user's answer is correct, their score is incremented by 10.
